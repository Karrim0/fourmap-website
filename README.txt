# FourMap — Engineering Services Platform

FourMap is a production-ready digital engineering services platform built for the Saudi market.

The platform enables individuals and companies to request engineering services online through a fully dynamic website connected to an admin dashboard and mobile application.

---

## Project Type
Client Project — Live Production Deployment

Live Website:
https://aaadosry.info/fourmap4/


## Overview

This project was built entirely from scratch without using any frameworks.

It includes:
- Multi-page public website
- Fully dynamic admin dashboard
- Database-driven content system
- Arabic-first RTL support
- Bilingual architecture (Arabic / English)

---

## Core Features

- Dynamic services management from admin panel
- Full content control without editing code
- Secure database interaction using PDO prepared statements
- Custom-built language switching system
- Responsive layout using CSS Grid and Flexbox
- WhatsApp integration per service
- Image lazy loading for performance optimization
- Modular file structure for maintainability

---

## Tech Stack

Backend:
- PHP 8
- MySQL
- PDO

Frontend:
- HTML5
- CSS3 (RTL Support)
- JavaScript

Server:
- Apache

---

## Architecture Notes

- Reusable layout components (header / footer includes)
- Separated database logic (db.php)
- Language files structured for scalability
- Admin panel designed for future feature expansion
- Secure form handling with server-side validation
